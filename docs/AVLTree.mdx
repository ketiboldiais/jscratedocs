## AVLTree

The `AVLTree` class implements an AVL tree. The AVL tree is a one type of
self-balancing binary search tree.

<AVLTree
	data={[
		{ c: 33, p: "" },
		{ c: 13, p: 33 },
		{ c: 53, p: 33 },
		{ c: 9, p: 13 },
		{ c: 21, p: 13 },
		{ c: 8, p: 9 },
		{ c: 11, p: 9 },
		{ c: null, p: 53, h: true },
		{ c: 61, p: 53 },
	]}
	cluster={true}
/>

## Constructor

To create a new `AVLTree` instance:

```javascript
import { AvlTree } from "@crates";
import { Print } from "@util";

const avl = AvlTree(20, 10, 50, 8, 15, 46, 60, 11, 32, 48);
Print(avl);
```
