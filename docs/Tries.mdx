# Trie

The `Trie` module implements a basic trie. Because tries consume
significant amounts of memory, the `Trie` module will only work with the
ASCII characters listed in the Alphabet dropwdown below (lower-case English
letters "a" through "z"). Uppercase letters are treated as lower-case
letters through a simple or-shortcircuit that checks for
`toLocaleLowerCase()`. If uppercase letters should be treated as unique
glyphs, see the next paragraph.

To include more glyphs in the alphabet, see the `addGlyph` and `addGlyphs`
methods below. Note, however, that tries achieve their time efficiency by
consuming large amounts of memory. With the default alphabet, each node in
the trie holds an array of length **26**. Extended ASCII support (including
all the control characters rarely used today) implies that each node stores
an array of **256**. In turn, this implies that, given an alphabet of 256
characters, a simple three letter word like "eat" results in three arrays
of size 256.

## Constructor

A `Trie` can be instantiated by passing an optional comma separated list of
string values to the `Trie()` function:

```javascript
import { Trie } from "@crates";
import { Print } from "@util";

const trie = Trie("Anne", "Zoe", "Cathy", "Hikari");
Print(trie);
```

```terminal
word: Anne
word: Cathy
word: Hikari
word: Zoe
```

## delete(ùëÜ)

Deletes the string ùëÜ in the trie:

```javascript
import { Trie } from "@crates";
import { Print } from "@util";

const trie = Trie("Billy", "Joe", "Cathy", "Hikari");

Print(trie);

trie.delete("Hikari");

Print(trie);
```

```terminal
word: Billy
word: Cathy
word: Hikari
word: Joe

word: Billy
word: Cathy
word: Joe
```

## has(ùëÜ)

Returns `true` if the trie contains the string ùëÜ, `false` otherwise.

```javascript
import { Trie } from "@crates";
import { Print } from "@util";

const trie = Trie("Anne", "Zoe", "Cathy", "Hikari");

const JoeVoted = trie.has("Joe");
const ZoeVoted = trie.has("Zoe");
const AnnVoted = trie.has("Ann");

Print(JoeVoted);
Print(ZoeVoted);
Print(AnnVoted);
```

```terminal
false
true
false
```

Note that the `has()` method returns the result of a search for the exact
match of ùëÜ. As such, the last query (`"Ann"`), returns false because there
is no word that exactly matches `"Ann"`.
