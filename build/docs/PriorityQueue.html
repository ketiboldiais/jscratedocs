PriorityQueue

The PriorityQueue module implements a basic priority queue, a special kind of queue. As a priority queue, each of the queue's elements are assumed to have a specific priority (from a finite set of possible priorities), and elements with higher priorities are processed before those with lower priorities. And as a queue, the elements are processed under FIFO â€” first in, first out. This is analogous to a queue of passengers boarding a flight: Boarding Group 1, Group 2, Group 3, etc.

NOTE: The PriorityQueue class only uses numbers to interpret priority levels, with smaller numbers signaling higher prority, and larger numbers signaling lower priority.
