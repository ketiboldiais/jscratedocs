# LinkedList

The `LinkedList` class implements a singly-linked list. Below, the variable
𝐿 corresponds to a `LinkedList` instance.

## 𝐿.len()

Returns the list's length.

## 𝐿.push(𝑉)

Inserts the value `𝑉` at the end of the list

## 𝐿.lop()

Removes the first element in the list

## 𝐿.prepend(𝑉)

Inserts `𝑉` at the head of the list

## 𝐿.first = 𝑉

Sets first element's value to `𝑉`

## 𝐿.first

Returns first list element's value (`null` if the list is empty)

## 𝐿.last = 𝑉

Sets the last element's value to `𝑉`

## 𝐿.last

Returns the last element's value to `𝑉` (`null` if the list is empty)

## 𝐿.pop()

Removes the last element in the list

## 𝐿.search(𝑉)

Returns the value `𝑉` in the list, otherwise `null`

## 𝐿.insert(𝑉, 𝑖)

Inserts the value `𝑉` at index `𝑖`

## 𝐿.remove(𝑖)

Removes the value `𝑉` at index `𝑖` (deleted element is returned)

## 𝐿.reverse(𝑖)

Mutates the list, where the list is reversed.

## 𝐿.reversed(𝑖)

Returns a new list with `𝐿`'s list reversed. `𝐿` is not mutated.

## 𝐿.arrayed()

Returns each element's value in an array.

## 𝐿.kreverse(𝑛)

Returns the list mutated, where every 𝑛 elements are reversed. If 𝑛 is 0,
1, or the length, no mutation occurs.
