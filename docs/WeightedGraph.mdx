The `WeightedGraph` module implements a weighted _simple graph_. That is, a
a simple graph with weighted edges.

## Constructor

A `WeightedGraph` can be instantiated by passing a comma separated list of
edges (or a spread of an array of edges). Alternatively, the parameter list
can be left empty, with edges and vertices added later via the `addEdge`
and `addVertices` methods.

```javascript
import { WeightedGraph } from "@crates";
import { Print } from "@util";

const wg = WeightedGraph(["A", "B", 4], ["A", "C", 2], ["B", "D", 3]);

Print(wg);
```

```javascript
	["B", "E", 3],
	["C", "D", 2],
	["C", "F", 4],
	["D", "E", 3],
	["D", "F", 1],
	["E", "F", 1],

Print(shortestPath_A_to_E);
```
