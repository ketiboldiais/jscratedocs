primeFactors(𝑛)​

Given an natural number 𝑛, returns an array of the prime factors of 𝑛. The array consists of tuples of the form:

[F,E][\texttt{F}, \texttt{E}][F,E]

where F{\texttt{F}}F is a prime factor, and E{\text{E}}E is the factor's exponent (i.e., how many times the factor occurs in 𝑛's prime factorization).

const n = primeFactors(315);
console.log(n);
[ [ 3, 2 ], [ 5, 1 ], [ 7, 1 ] ]
metricdata
time complexityO(n){O(\sqrt{n})}O(n​)

The prime factors of a number n{n}n are the factors n{n}n that are prime. For example, for the number 315,{315,}315, the prime factors are 3,5,7.{3,5,7.}3,5,7. A naive approach to finding the prime factors is to iterate from i=0{i=0}i=0 to n.{n.}n. If i{i}i is prime, then we check if it divides n.{n.}n. If it does, we divide n{n}n by i,{i,}i, and save i{i}i as a prime factor. We keep i{i}i at that prime until we find that it cannot divide the new n.{n.}n. We continue doing so until either we've either reduced n{n}n to 1,{1,}1, or until we've reached n:{n:}n:

1end

Once we've reduced the number to 1, we've found the primes:

[2,5][2,5][2,5]

In pseudocode:

    let i = 2;
    let primes = [];
    let len = 0; // the current length of the primes

A far better approach is recognizing that we only need to check for potential primes up to n.{\sqrt{n}.}n​. This stems from the fact that a prime number n{n}n takes the form n=a⋅b,{n = a \cdot b,}n=a⋅b, where n/a{n/a}n/a yields a remainder of zero (assuming a{a}a is the smaller number; if b{b}b were the smaller, the definition remains: n/b{n/b}n/b yields a remainder of zero). From this definition, it must be true that a{a}a lies on the interval 2≤a<n−a{2 \leq a \lt n-a}2≤a<n−a (otherwise a{a}a woulnd't be a factor). If we suppose that b=na,{b = \dfrac{n}{a},}b=an​, then a≤b.{a \leq b.}a≤b. And if a≤b,{a \leq b,}a≤b, it must be true that:

a⋅b≥a2n≥a2n≥a\begin{aligned} a \cdot b &\geq a^2 \\[1em] n &\geq a^2 \\[1em] \sqrt{n} &\geq a \end{aligned}a⋅bnn​​≥a2≥a2≥a​

Hence, it must be the case that in our code, our variant i{i}i (the number we're checking as a prime factor) must be less than n:{\sqrt{n}:}n​:
