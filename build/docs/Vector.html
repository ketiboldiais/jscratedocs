Vector

The Vector namespace provides the following containers:

  • Vector.int8
  • Vector.uint8
  • Vector.uint8Clamped
  • Vector.int16
  • Vector.int32
  • Vector.uint32
  • Vector.float32
  • Vector.float64
  • Vector.bigint64
  • Vector.biguint64

Each of these submodules implements a typed dynamic array. The submodules map to JavaScript's typed arrays. By default, typed arrays in JavaScript have a set size once instantiated. The Vector module provides a way to use the typed arrays as dynamic arrays.

For most applications, JavaScript's usual array object suffices. The Vector submodules are mainly used as auxiliary data structures for some of jsCrate's internals and testing suites, but are provided here for public availability.

All instances Vector have the API listed below, with a few caveats, highlighted accordingly.

Constructor‚Äã

To instantiate a vector, import the Vector namespace and call the constructor. An optional comma separated list of values can be passed to initialize the vector immediately:

import { Vector } from "@crates";

const int8 = Vector.int8(1, -2, -3, 4, 5);
const uint8 = Vector.uint8(0, 1, 7, 4, 9);
const uint8c = Vector.uint8Clamped(9, 2, 8, 4, 7);
const int16 = Vector.int16(39, 19, 72, 21, 55);
const int32 = Vector.int32(9, 12, 13, 4, 25);
const uint32 = Vector.uint32(35, 210, 313, 215, 512);
const float32 = Vector.float32(1.1, 2.8, 3.7, 4.9, 5.1);
const float64 = Vector.float64(34.8, 9.8, 3.7, 4.6, 5.2);
const bigint64 = Vector.bigint64(1n, 5n, 7n, 8n, 9n);
const biguint64 = Vector.biguint64(12n, 17n, 10n, 15n);

console.log(int8);
console.log(uint8);
console.log(uint8c);
console.log(int16);
console.log(int32);
console.log(uint32);
console.log(float32);
console.log(float64);
console.log(bigint64);
console.log(biguint64);

ùëâ.pairSum(ùëÜ)‚Äã

Returns a new vector of type ùëâ containing the numbers that sum to ùëÜ. If no such pair exists, null is returned.

import { Vector } from "@crates";
import { Print } from "@util";

const vect = Vector.int8(10, 5, 2, 3, -6, 9, 11);

const pairSum = vect.pairSum(4);

Print(pairSum);
[10,-6]

ùëâ.maxBandLength()‚Äã

Returns the size of the largest subset in the vector containing consecutive numbers.

import { Vector } from "@crates";
import { Print } from "@util";

const vect = Vector.int8(1, 9, 3, 0, 18, 5, 2, 4, 10, 7, 12, 6);

const longestBandLength = vect.maxBandLength();
Print(longestBandLength);
8

In the example above, the largest subset is:

{0,1,2,3,4,5,6,7}\{ 0,1,2,3,4,5,6,7 \}{0,1,2,3,4,5,6,7}
