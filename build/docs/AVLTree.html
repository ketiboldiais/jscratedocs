AVLTree

AVLTree​

The AVLTree class implements an AVL tree. The AVL tree is a one type of self-balancing binary search tree.

Constructor​

To create a new AVLTree instance:

import { AvlTree } from "@crates";
import { Print } from "@util";

const avl = AvlTree(20, 10, 50, 8, 15, 46, 60, 11, 32, 48);
Print(avl);
